Git status - какие файлы в каком состояние находятся
Git status -s - упрощенный вывод
Git add . - добавляет изменения рабочего каталога в проиндексированный файл
Git commit - берет все файлы и добавляет в индекс
Git commit -m “добавить коментарий”
Git init - создает новый репозиторий гит
Ls -la - показывает информацию о файлах в индексе и рабочем дереве
Git show - отображает объект в простом виде
Git log - перечисляет коммиты, сделанные в репозитории в обратном и хронологическом порядке
Git checkout -b «branch 1» - создание ветки
Git checkout main - переход из ветки в основной
git remote -v - в каком репозитории сейчас
git merge branch 1 - перенос данных из ветки в основную
Gut stash pop- преостановить работу
Git diff - показывает все незакомеченные файл (исп. для вычислений разницы между любыми двумя деревьями)
Git restore - операция отмены
Git push - исп. для выгрузки локального репозитория в удаленный
Git remote add «…» - добавить удаленный репозиторий и присвоить ему имя
